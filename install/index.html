Step 1 - Intro

Please have this information at hand:

  1. Database name
  2. Database username
  3. Database password
  4. Database host

This installation script will generate the config.php file in your root script directory. If it's unable to create it, then open config-manual.php in your favorite text editor and modify the changes yourself. After you have finished, rename it to config.php. Are you ready?

Step 2 - Database

On the right side you should enter your database connection details. If you're not sure about these, contact your host.

Step 3 - Contest

Fill in the required information about your contest. You will be able to change these later in the administration panel.
